Steps to use this program

Step 1:
Download the latest version from https://github.com/EdwinVanRooij/komplete-darkness/releases

Step 2:
Extract all files.

Step 3:
Run the setup, answer the prompted questions.

Step 4:
Copy the generated config.json to the main folder.

Step 5:
Run the main program.

If anything was unclear, have a look at the demo video:
https://youtu.be/kYX7PS1GVQs
